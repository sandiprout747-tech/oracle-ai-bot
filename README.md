# 🤖 Oracle AI Bot

> A powerful personal Telegram AI bot with PC control, notes, todos, voice, vision, email and more.  
> **One command to deploy on any Windows laptop.**

Created by **Sandip** · [@sandiprout747-tech](https://github.com/sandiprout747-tech)

---

## 🐧 Deploy in One Command

Open CMD and paste:
```cmd
curl -o install.py https://raw.githubusercontent.com/sandiprout747-tech/oracle-ai-bot/main/install.py && python install.py
```

That's it. The penguin installer handles everything.

---

## ✦ What It Does

- ✅ Checks Python 3.8+
- ✅ Installs all packages automatically
- ✅ Creates all folders
- ✅ Downloads bot files from GitHub
- ✅ Asks 6 questions to personalize your bot
- ✅ Creates StartAI.bat on your Desktop
- ✅ Launches both bots

Then open Telegram and send `/status` — you're live.
